TURBOSAW.COM LAST OPPORTUNITY EWAHUNTING.COM

I became the interim website administrator in January 2016. The site has been redone since I have left, the link goes to the archived version.
Last Opportunity was the end product of the 168-hr Code Jam for the 2019 Computer Simulation and Graphics Conference at the University of Tulsa.
My primary responsibilities were programming many of the visual aspects of the game including:

  • Shaders
  • Post-Processing Effects
  • Controllers
  • UI Elements

The languages used were C# and HLSL in Unity.
This website was built for a local hunting outfitter.
